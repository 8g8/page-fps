# page-fps

> Inject fps indicator into your page.



## Install

> npm i --save page-fps



## Usage


 - code usage

```js
import PF from 'page-fps';

const pf = new PF()
pf.start();
pf.stop();
```


 - script usage

Just import `dist/entry.min.js` into your document with script tag.
